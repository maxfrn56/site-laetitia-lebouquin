# Site vitrine - Laetitia Le Bouquin

Site vitrine pour Laetitia Le Bouquin, professionnelle du massage bien-être et de la nutrition-santé.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript vanilla

## Structure du projet

- `index.html` - Page principale (one-page)
- `style.css` - Styles CSS
- `images/` - Dossier contenant toutes les images

## Déploiement

Le site est déployé sur Netlify.
